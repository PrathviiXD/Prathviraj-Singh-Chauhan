### Projects

#### BiasRadar
**Challenge:** In an era of increasing data-driven decisions, detecting and mitigating bias in datasets and algorithms has become crucial to ensure fairness and ethical AI practices.  
**Solution:** Developed BiasRadar, an AI-powered tool that analyzes datasets for potential biases using machine learning algorithms to identify patterns of discrimination and provide actionable recommendations for mitigation.  
**Impact:** The project successfully reduced bias detection time by 70% and improved model fairness, contributing to more equitable AI applications in various domains.  
**Tech Stack:** Python, TensorFlow, Pandas, Scikit-learn, Jupyter Notebook.  
*AI Integration:* Utilized advanced machine learning models for bias detection, incorporating explainable AI techniques to provide transparent insights.

#### Gift Shop E-Commerce Web
**Challenge:** Small businesses struggle with establishing an online presence, leading to lost sales opportunities and inefficient inventory management.  
**Solution:** Built a full-featured e-commerce website for a gift shop, implementing user authentication, product catalog, shopping cart, and payment integration to create a seamless online shopping experience.  
**Impact:** Increased the gift shop's online sales by 50% within the first quarter, streamlined operations, and enhanced customer satisfaction through intuitive design and fast checkout.  
**Tech Stack:** HTML, CSS, JavaScript, Node.js, Express.js, MongoDB, Stripe API.  
*Advanced Logic:* Implemented dynamic pricing algorithms and inventory tracking system to optimize stock levels and pricing strategies.
